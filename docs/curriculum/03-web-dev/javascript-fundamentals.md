# JavaScript Fundamentals

## Required Hours
- **Total Hours**: 25 hours
- **Session Pattern**: 6 sessions of 4 hours, plus guided projects

## Purpose

JavaScript brings interactivity to web pages, enabling dynamic user experiences and client-side logic. As the primary programming language of the web, JavaScript skills are essential for modern web development and extend to server-side development with Node.js.

## Learning Outcomes

Students will be able to:

- Write JavaScript to add interactivity to web pages
- Manipulate the Document Object Model (DOM) effectively
- Handle user events and respond appropriately
- Work with asynchronous operations
- Apply JavaScript best practices and modern syntax

## Core Content

### JavaScript Basics

Core language features:

**Variables and Types**:
- var, let, and const
- Primitive types: string, number, boolean, null, undefined
- Type coercion and comparison
- Template literals for strings
- Modern variable declaration practices

**Functions**:
- Function declarations and expressions
- Arrow functions
- Parameters and return values
- Default parameters
- Higher-order functions
- Closures and scope

**Objects and Arrays**:
- Object literals and properties
- Array creation and manipulation
- Array methods: map, filter, reduce, forEach
- Destructuring assignment
- Spread and rest operators

### DOM Manipulation

Interacting with web pages:

**Selecting Elements**:
- getElementById, querySelector, querySelectorAll
- getElementsByClassName and getElementsByTagName
- Traversing the DOM tree
- NodeList vs HTMLCollection

**Modifying Content**:
- textContent and innerHTML
- Creating new elements
- Appending and removing elements
- Modifying attributes
- Working with classes: add, remove, toggle

**Styling with JavaScript**:
- Inline styles via style property
- Getting computed styles
- Modifying CSS classes
- Dynamic styling based on conditions

### Event Handling

Responding to user interactions:

**Event Types**:
- Click, submit, input, change
- Mouse events: mouseover, mouseout, mousemove
- Keyboard events: keydown, keyup, keypress
- Form events and validation
- Load and DOMContentLoaded

**Event Listeners**:
- addEventListener and removeEventListener
- Event objects and properties
- Event bubbling and capturing
- Preventing default behavior
- Event delegation

### Asynchronous JavaScript

Handling delayed operations:

**Callbacks**:
- Function as arguments
- Callback patterns
- Callback hell and how to avoid it

**Promises**:
- Creating and consuming promises
- then and catch methods
- Promise chaining
- Error handling

**Async/Await**:
- Modern asynchronous syntax
- Error handling with try/catch
- Sequential vs parallel operations
- Practical async patterns

### Working with APIs

Fetching external data:

**Fetch API**:
- Making HTTP requests
- Handling responses
- JSON parsing
- Error handling
- HTTP methods: GET, POST, PUT, DELETE

**Working with JSON**:
- JSON syntax and structure
- JSON.parse and JSON.stringify
- Common data formats
- Displaying API data in the DOM

### Modern JavaScript

ES6+ features and best practices:

**Modern Syntax**:
- let and const over var
- Arrow functions
- Template literals
- Destructuring
- Default parameters
- Spread/rest operators

**Modules** (Introduction):
- Import and export
- Module scope
- Organizing code into modules

**Best Practices**:
- Use strict mode
- Avoid global variables
- Consistent naming conventions
- Error handling
- Code organization

### Browser Developer Tools

Debugging JavaScript:

- Console for logging and debugging
- Breakpoints and step-through debugging
- Network tab for API calls
- Elements tab for DOM inspection
- Performance profiling basics

## Deliverables

- Interactive web pages with JavaScript functionality
- Form validation with custom error messages
- Application consuming external API data
- Event-driven interactive game or tool
- Portfolio demonstrating modern JavaScript features
