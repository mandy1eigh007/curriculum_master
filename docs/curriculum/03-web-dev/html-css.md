# HTML and CSS

## Required Hours
- **Total Hours**: 20 hours
- **Session Pattern**: 5 sessions of 4 hours each

## Purpose

HTML and CSS are the foundation of web development. HTML provides structure and content, while CSS controls presentation and styling. Together, they enable developers to create accessible, visually appealing web pages that work across different devices and browsers.

## Learning Outcomes

Students will be able to:

- Write semantic HTML that structures content appropriately
- Apply CSS to style web pages professionally
- Understand the box model and layout fundamentals
- Create accessible web content following best practices
- Debug HTML and CSS issues effectively

## Core Content

### HTML Fundamentals

Structuring web content:

**Document Structure**:
- DOCTYPE declaration
- HTML, head, and body elements
- Meta tags for character encoding and viewport
- Title and semantic importance

**Semantic Elements**:
- Headers: h1 through h6
- Paragraphs, lists (ordered and unordered)
- Links and navigation
- Articles, sections, headers, footers
- Main, aside, and figure elements

**Forms and Input**:
- Form element and submission
- Input types: text, email, password, number, date
- Textareas and select dropdowns
- Labels for accessibility
- Buttons and form validation

**Media and Embedding**:
- Images with alt text for accessibility
- Audio and video elements
- Embedding content with iframe
- Responsive images with srcset

### CSS Fundamentals

Styling web content:

**Selectors and Specificity**:
- Element, class, and ID selectors
- Descendant and child selectors
- Pseudo-classes and pseudo-elements
- Selector specificity and the cascade
- Inheritance of properties

**Box Model**:
- Content, padding, border, margin
- Box-sizing: content-box vs border-box
- Width, height, and sizing
- Display property: block, inline, inline-block
- Overflow handling

**Typography**:
- Font families and web-safe fonts
- Font size, weight, and style
- Line height and letter spacing
- Text alignment and decoration
- Web fonts and font loading

**Colors and Backgrounds**:
- Color values: names, hex, RGB, RGBA, HSL
- Background colors and images
- Gradients: linear and radial
- Background positioning and sizing
- Transparency and opacity

### Layout Techniques

Positioning elements on the page:

**Positioning**:
- Static (default) positioning
- Relative positioning
- Absolute positioning
- Fixed positioning
- Sticky positioning
- Z-index and stacking context

**Flexbox**:
- Flex containers and items
- Main axis and cross axis
- Justify-content and align-items
- Flex-direction, flex-wrap
- Flex-grow, flex-shrink, flex-basis
- Common layout patterns

**Grid** (Introduction):
- Grid containers and items
- Rows and columns
- Grid template areas
- Gaps and spacing
- Responsive grid layouts

### Responsive Design Basics

Creating layouts that work on all devices:

- **Mobile-First Approach**: Start with mobile, enhance for desktop
- **Media Queries**: Conditional CSS for different screen sizes
- **Viewport Meta Tag**: Proper mobile rendering
- **Flexible Units**: %, em, rem, vw, vh
- **Responsive Images**: Scaling and resolution

### Accessibility

Making content usable for everyone:

- **Semantic HTML**: Using appropriate elements
- **Alt Text**: Describing images for screen readers
- **Keyboard Navigation**: Tab order and focus
- **Color Contrast**: Readable text
- **ARIA Attributes**: When semantic HTML isn't enough
- **Form Labels**: Connecting labels to inputs

## Deliverables

- Multi-page website with semantic HTML structure
- Styled pages demonstrating CSS layout techniques
- Responsive design adapting to different screen sizes
- Accessible web content meeting WCAG guidelines
- Form with proper validation and user feedback
