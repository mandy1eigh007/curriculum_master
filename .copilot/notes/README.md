# Copilot Notes

This folder is for sharing information between you and Copilot.

## Usage

- Drop any text files, notes, or documentation here that you want Copilot to read
- Files in this folder will be ignored by git (added to .gitignore)
- Use this for context, requirements, or any information you want to share

## Files

Add your files here and mention them in PR comments for Copilot to review.
